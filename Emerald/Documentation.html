Introduction First we need to tell you "Thank you for your purchase!"
Hope you'll be appreciated by our work. Thank you again. Theme Installation You need to install FileZilla first to transfer files to your MyBB directory via FTP
Connect to your website via FTP using FileZilla, and copy all files of the "Upload" directory of the provided folder (after unzipping the archive) when you purchased the item.
Go to your MyBB forum admin panel, open the "Templates & Style" page, then click in "Import a theme". Choose "Local File" and select the "theme.xml" file from the XML directory of the theme's provided folder. Choose any name for your theme on the next input. Then click on "Import Theme".
To set your theme as a default one for your forum, go again to "Templates & Style" and click on the little window icon.
Logo Changing First, go to your AdminCP, go to "Templates" page and select the templates group of the theme >> Header Templates >> header :

You'll find the HTML/CSS logo of the theme which is : <i class="fa fa-diamond"></i> Emerald
To change that into an image (which is highly not recommended because of margin & height problems), replace that to : <img src="YOUR IMAGE LINK" />
To keep an HTML/CSS logo, change into any text, for example "BestForums". If you want to add a font icon for that (as our logo) please check here. For example <i class="fa fa-diamond"></i> means diamond icon. Header Block Go to your AdminCP, go to "Templates" page and select the templates group of the theme >> Header Templates >> header :

You'll find the title which is : Emerald - Demo
And the description : A demo forum for a great theme
You just have to change the text into whatever you want, also add html stuffs if you have some skills. Sidebars Go to Templates >> Emerald Templates >> Index Page Templates >> index
To delete the sidebars, search for : <div class="forum" style="float: left;">{$forums}{$boardstats}</div> <div class="side" style="float: right; width: 22%; margin: auto;">
And replace it with : <div class="forum">{$forums}{$boardstats}</div> <div class="side" style="display:none;">
To delete the announcement sidebar, search for (and then delete it) : <table style="" class="tborder" border="0" cellpadding="{$theme['tablespace']}" cellspacing="{$theme['borderwidth']}"> <tbody><tr> <td style="font-weight: bold;" class="thead"> <i class="fa fa-bullhorn"></i> Announcements </td> </tr> <tr> <td class="trow2" style="padding:11px"> <a href="#">This is an example !</a> </td> </tr> </tbody></table>
To add an announcement (with link), search for the past code, the first link is : <a href="#">This is an example !</a>, as a first link you can edit the text and replace "#" with your link.
To add another link, add after that (for example) : <br><a href="YOUR LINK">THE TEXT</a> Font Icons If you like to add an icon to your text logo (as wrote at the section "Logo Changing"):
First, search for an icon here : http://fortawesome.github.io/Font-Awesome/icons/
For example i click on "diamond" icon, i'll be redirected to http://fortawesome.github.io/Font-Awesome/icon/diamond/
I will get this code : <i class="fa fa-diamond"></i> It represents the icon i want.
Then, i can use it for my logo (as wrote before) like this : <i class="fa fa-diamond"></i> Emerald
